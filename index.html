PolyEditor

Instructions

Select a base shape and apply one or more operations to it. You can also input a polyhedron directly on the box on the top right, which uses Conway polyhedron notation: The rightmost (last) letter represents the starting polyhedron, then operations are applied in order, going from right to left.

You can zoom and rotate the polyhedron using the mouse, or use the controls under the viewer to modify the visualization.

Base shapes

  • Tetrahedron
  • Cube
  • Octahedron
  • Dodecahedron
  • Icosahedron

Operations

  • Ambo
  • Bevel
  • Dual
  • Expand
  • Gyro
  • Join
  • Kis
  • Meta
  • Needle
  • Ortho
  • Snub
  • Truncate
  • Zip
