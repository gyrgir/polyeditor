PolyEditor

A polyhedron visualizer using three.js.

Describe a polyhedron using Conway polyhedron notation to draw it. The rightmost (last) letter represents the starting polyhedron, on which a series of operations are applied in order, going from right to left.

Base shapes

  • T tetrahedron
  • C cube
  • O octahedron
  • D dodecahedron
  • I icosahedron

Supported operations

  • a ambo
  • b bevel
  • d dual
  • e expand
  • j join
  • k kis
  • m meta
  • n needle
  • o ortho
  • t truncate
  • z zip
